# jamotong 한자 훈음(뜻·음) 표 — 음(가나다) 순
#
# [출처/저작권]
# 이 표는 jamotong 프로젝트가 직접 작성한 큐레이션이다. 한자의 전통적 대표 훈음(예: 家=집 가)은
# 수백 년간 통용된 언어적 사실로 개별 항목은 저작권 보호 대상이 아니며, 이 편집물은 특정
# 사전·서적을 복사하지 않고 일반 지식으로부터 새로 작성했다.
#
# 수록 범위(2단계):
#   (1) 교육부 '한문 교육용 기초 한자'(1,800자) 수준의 상용 한자 — 초판 큐레이션.
#   (2) 그 밖의 변환 테이블 수록 한자(주로 인명용) 중 대표 훈음을 확정할 수 있는 것 —
#       Unicode Unihan Database 의 영문 정의(kDefinition, Unicode License)를 근거로 각 한자의
#       한국어 대표 훈음을 새로 작성하고, 음(音)이 변환 테이블과 일치하는지 기계 검증해 수록했다.
#       뜻이 모호하거나 대표 훈을 확정하기 어려운 벽자는 넣지 않고 음(音) 폴백에 맡긴다
#       (후보창이 훈음 없는 한자는 음만 표시).
#
# 훈음은 널리 쓰이는 대표형 하나만 실었다(다음자·복수 훈은 대표 1개). 오류·보완 제보 환영.
#
